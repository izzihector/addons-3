Odoo property Sale, Rental and Tenancy Management

This odoo property management application is complete solutions for manage all kind of real estate property. this application provide features to allow to your landlord to create property for sale and rent. Tenant will check out available property and reserve for rent and buy from their login. Agent can checkout their commission related to the property and tenancy agreement. This application also helps to settlement for multiple commission for agent.

Tenant Flow

There are two ways to Buy/Rent any property

First Way : Login with Tenant User and you can see all the available property

Open any property which you want to buy/rent. You can see "Reserve for Rent" button if property is for rent and "Buy" button if property is for Buy.

If you click on "Reserve For Rent" Or "Buy" button it will open Tenancy Agreement wizard to enter some informations and show you some calculations.

If you have varified the informations and calculations, you can click on confirm agreement. You can see all your agreement here.

Second Way: Go to Tenancy Agreement and click on create. You can select only avaialble property and some other required details only. You can also see all the calculations there and if you varified everything then you can confirm the agreement.

Confirmed Tenancy Agreement

You can see status of the property is changed

Invoicing for rent and buy

For Rented Property only: We have set automated Actions (Cron) for check agreement start and expiry status. you can see agreement is already running and generate invoice for that agreement.

Automatically Generated Invoices for That Agreement

For Buy Property only: You can generate invoice manually. There is one button called "Create Invoice" in tenancy agreement

It will Generate Invoices based on selected installments and payment options for That Agreement

Agent Flow

Agent can check their commission lines. whenever invoice will generate for buy or rent property it will generate commission line for that agent.

Agent or Admin can settle commission for perticular agent

Once you confirm the commission settlement, you can generate one vendor bill for that agent.
